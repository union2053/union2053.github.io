Question

What's wrong with this attempted declaration of a character string? 

int main(void)
{
   char name[] = {'L', 'e', 's', 's' };
  ...
}

The initialization should include a '\0' if you want the result to be a string.

Of course, the alternative syntax adds the null character automatically: 

char name[] = "Less";

Question 2

    puts(++ptr);
