C Dependent nested loops

PreviousNext

You can make the inner loop behave differently each cycle by making part of the inner loop depend on the outer loop.

The following code makes the starting character of the inner loop depend on the cycle number of the outer loop.

It also uses the newer comment style and const instead of #define to help you get comfortable with both approaches. 

// rows2.c -- using 
#include <stdio.h>
int main(void)
{
    const int ROWS = 6;
    const int CHARS = 6;
    int row;
    char ch;

    for (row = 0; row < ROWS; row++)
    {
        for (ch = ('A' + row); ch < ('A' + CHARS); ch++)
            printf("%c", ch);
        printf("\n");
    }

    return 0;
}

Here's the output this time:

Because row is added to 'A' during each cycle of the outer loop, ch is initialized in each row to one character later in the alphabet.

The test condition, however, is unaltered, so each row still ends on F.

This results in one fewer character being printed in each row.
