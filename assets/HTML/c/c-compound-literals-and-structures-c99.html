C Compound Literals and Structures (C99)

PreviousNext

C99's compound literal feature is available for structures as well as for arrays.

It's handy if you just need a temporary structure value.

For instance, you can use compound literals to create a structure to be used as a function argument or to be assigned to another structure.

The following code shows an example using compound literals to provide two alternative values for a structure variable. 

#include <stdio.h>
#define MAXTITL 41
#define MAXAUTL 31

struct book {          // structure template: tag is book
    char title[MAXTITL];
    char author[MAXAUTL];
    float value;
};

int main(void)
{
    struct book readfirst;
    int score;

    readfirst = (struct book) {"Tutorial for CSS",
                               "demo2s.com",
                               1.25};
    printf("%s by %s: $%.2f\n",readfirst.title,
           readfirst.author, readfirst.value);

    return 0;
}
