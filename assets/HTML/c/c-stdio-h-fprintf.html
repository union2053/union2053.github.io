C stdio.h fprintf

PreviousNext

Example

The following code saves formatted text to a text file. 

#include <stdio.h>

int main ()
{
   FILE * pFile;
   int n;
   char name [100]= "test test";

   pFile = fopen ("main.c","w");
   for (n=0 ; n<3 ; n++)
   {
     fprintf (pFile, "Name %d [%-10.10s]\n",n+1,name);
   }
   fclose (pFile);

   return 0;
}

Output:

Prototype 

int fprintf ( FILE * stream, const char * format, ... );

Description

fprintf() can write formatted data to a file stream.

It uses the same formatting string as printf().

Parameters

Item
 Description
stream
 Pointer to a FILE object that identifies an output stream.
format
 C string that contains the text to be written to the stream.

Return Value

On success, the total number of characters written is returned.

If a writing error occurs, the error indicator (ferror) is set and a negative number is returned.

If a multibyte character encoding error occurs while writing wide characters, errno is set to EILSEQ and a negative number is returned.
