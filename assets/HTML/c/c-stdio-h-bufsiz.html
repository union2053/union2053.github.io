Example

The following code prints out the value of BUFSIZ. 

#include <stdio.h>      /* printf */

int main ()
{
  printf ("BUFSIZ is %d \n", BUFSIZ);
  return 0;
}

Description

This macro constant defines an integral expression with the size of the buffer used by the setbuf() function.
