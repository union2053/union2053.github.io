C Passing Structure Members to a function

Passing structure members as arguments

As long as a structure member is a data type with a single value (that is, an int or one of its relatives, a char, a float, a double, or a pointer), it can be passed as a function argument to a function that accepts that particular type. 

#include <stdio.h>
#define FUNDLEN 50

struct funds {
    char   bank[FUNDLEN];
    double bankfund;
    char   save[FUNDLEN];
    double savefund;
};

double sum(double, double);

int main(void)
{
    struct funds jack = {
        "Bank A",
        4.27,
        "Loan",
        8.94
    };

    printf("Jack has a total of $%.2f.\n",
           sum(jack.bankfund, jack.savefund) );
    return 0;
}

/* adds two double numbers */
double sum(double x, double y)
{
    return(x + y);
}

Using the Structure Address

Passing a pointer to a structure 

#include <stdio.h>
#define FUNDLEN 50

struct funds {
    char   bank[FUNDLEN];
    double bankfund;
    char   save[FUNDLEN];
    double savefund;
};

double sum(const struct funds *);      /* argument is a pointer */

int main(void)
{
    struct funds jack = {
        "Bank A",
        4.27,
        "Loan",
        8.94
    };

    printf("Jack has a total of $%.2f.\n", sum(&jack));

    return 0;
}

double sum(const struct funds * money)
{
    return(money->bankfund + money->savefund);
}

Output:

The sum() function uses a pointer (money) to a funds structure for its single argument.

Passing the address &jack to the function causes the pointer money to point to the structure jack.

Then the -> operator is used to gain the values of jack.bankfund and jack.savefund.

Because the function does not alter the contents of the pointed-to value, it declares money as a pointer-to-const.

Passing a Structure as an Argument

Passing a structure 

#include <stdio.h>
#define FUNDLEN 50

struct funds {
    char   bank[FUNDLEN];
    double bankfund;
    char   save[FUNDLEN];
    double savefund;
};

double sum(struct funds rec1);     /* argument is a structure */

int main(void)
{
    struct funds jack = {
        "Bank A",
        4.27,
        "Loan",
        8.94
    };

    printf("Jack has a total of $%.2f.\n", sum(jack));

    return 0;
}

double sum(struct funds rec1)
{
    return(rec1.bankfund + rec1.savefund);
}

Output:

We replaced money, which was a pointer to struct funds, with rec1, which is a struct funds variable.

When sum() is called, an automatic variable called rec1 is created according to the funds template.

The members of this structure are then initialized to be copies of the values held in the corresponding members of the structure jack.
