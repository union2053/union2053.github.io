C Checking for C11/C17 Support for secure string processing functions

PreviousNext

The default set of string processing functions that the standard library provides are not secure.

They provide considerable potential for errors in your code.

These problems can occur because there is no verification that arrays are large enough for the operations.

For these reasons, the C17 standard includes optional versions of the string processing functions that are more secure and less prone to errors because the dimensions of arrays are checked.

All the new functions have names that end with _s.

C11/C17 Support

We determine whether the standard library that comes with your C compiler supports these optional functions.

Just compile and execute the following code: 

#include <stdio.h> 

int main(void) 
{ 
        //this line shouldn't compile in MS Visual Studio, 
        //comment it if necessary 
        printf("__STDC_VERSION__ = %d\n", __STDC_VERSION__); 

      #if defined __STDC_LIB_EXT1__ 
        printf("Optional functions are defined.\n"); 
      #else 
        printf("Optional functions are not defined.\n"); 
      #endif 
        return 0; 
}

The __STDC_VERSION__ macro should yield value 201710L or 201112L for C17 and C11 versions, respectively.

A compiler that implements the optional functions according to the C11/C17 standard will define the symbol __STDC_LIB_EXT1__ (C Library Extension 1).

This code uses preprocessor directives to insert one of two printf() statements, depending on whether the symbol __STDC_LIB_EXT1__ is defined.

If this symbol is defined, the code will output the message

If the symbol __STDC_LIB_EXT1__ is not defined, you will see the following output:

To use the optional functions in string.h, you must define the __STDC_WANT_LIB_EXT1__ symbol in your source file to represent the value 1, prior to the include directive for string.h, like this: 

// Make optional versions of functions available 
#define __STDC_WANT_LIB_EXT1__ 1  
// Header for string functions 
#include <string.h>

If you don't define this symbol as 1, only the standard set of string processing functions will be available.
