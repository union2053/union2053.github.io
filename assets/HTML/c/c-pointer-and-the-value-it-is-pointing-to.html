C Pointer and the value it is pointing to

PreviousNext

The following image illustrates using a pointer.

The pointer is type char*, which is a pointer to char.

The pChar variable can only store addresses of variables of type char.

The value stored in ch is changed through the pointer.

1. Create a pointer variable: 

char* pChar = NULL;

The pChar, which is a variable, is stored in address of 12ff00.


2. Create a variable of type char: 

char ch = 0;

ch is a char variable and stored at address of 12ff0f.


3. Store the address of ch in pChar: 

pChar = &ch;

We get and address of ch and assign it to pChar.


4. Indirectly store a value in ch: 

*pChar = 'a';

Here we changed the value of ch via its address which is stored in pChar.
