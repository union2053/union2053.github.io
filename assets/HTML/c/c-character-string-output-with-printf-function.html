C Character String output with printf() function

PreviousNext

printf() takes a string address as an argument.

printf() does not automatically print each string on a new line.

Instead, you must indicate where you want new lines. Therefore, 

printf("%s\n", string);

has the same effect as 

puts(string);

printf() makes it simple to combine strings for one line of printing.

For example, the following statement combines Hi, with the user's name and a #defined character string, all on one line: 

printf("Well, %s, %s\n", name, MSG);
