Question 1

Here are groups of one or more macros followed by a source code line that uses them.

What code results in each case?

Is it valid code? 

a.
     #define FPM 5280       /* feet per mile */
     dist = FPM * miles;

b.
     #define FEET 4
     #define POD FEET + FEET
     variable1 = FEET * POD;

c.
     #define SIX = 6;
     nex = SIX;

d.
     #define NEW(X) X + 5
     y = NEW(y);
     variable2 = NEW(variable2) * lob;
     est = NEW(variable2) / NEW(y);
     variable3 = lob * NEW(-variable2);



a. dist = 5280 * miles;  is valid.

b.   variable1 = 4 * 4 + 4;  is valid. 

c.  nex = = 6;;  is invalid. 

If there were no space between the two equal signs, 
it would be valid but useless.

The user forgot that he or she was writing
for the preprocessor, not writing in C.

d. y = y + 5;  is valid.  
variable2 = variable2 + 5 * lob;  is valid, 
but this is probably not the desired result. 

est = variable2 + 5/ y + 5;  is valid, but this is probably not
the desired result. 
variable3 = lob *-variable2 + 5;  is valid, 
but this is probably not the desired result.

Question 2

Define a macro function that returns the minimum of two values. 


 #define MIN(X,Y) ( (X) < (Y) ? (X) : (Y) )

Question 3

Define the EVEN_GT(X,Y) macro, which returns 1 if X is even and also greater than Y. 


#define EVEN_GT(X,Y) ( (X) > (Y) && (X) % 2 == 0 ? 1 : 0 )

Question 4

Define a macro function that prints the representations and the values of two integer expressions.

For example, it might print 

3+4 is 7 and 4*12 is 48

if its arguments are 3+4 and 4*12.



#define PR(X,Y) printf(#X " is %d and " #Y " is %d\n", X,Y)
               
Because X  and  Y  are never exposed to any other operations 
(such as multiplication) in
this macro, you don't have to cocoon everything in parentheses.

Question 5

Create #define statements to accomplish the following goals: 

a. Create a named constant of value 25.

b. Have SPACE represent the space character.

c. Have PS() represent printing the space character.

d. Have BIG(X) represent adding 3 to X.

e. Have SUMSQ(X,Y) represent the sums of the squares of X and Y.


a. #define QUARTERCENTURY 25

b. #define SPACE ' '

c. #define PS() putchar(' ')

   or

   #define PS() putchar(SPACE)

d. #define BIG(X) ((X) + 3)

e. #define SUMSQ(X,Y) ((X)*(X) + (Y)*(Y))
