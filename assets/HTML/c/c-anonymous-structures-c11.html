C Anonymous Structures (C11)

PreviousNext

An anonymous structure is a structure member that is an unnamed structure.

To see how this works, first consider the following setup for a nested structure: 

struct names
{
    char first[20];
    char last[20];
};
struct person
{
    int id;
    struct names name; // nested structure member
};
struct person jack = {123, {"James", "Bound"}};

Here the name member is a nested structure, and you could use an expression like jack.name. first to access "Ted": 

puts(jack.name.first);

With C11, you can define person using a nested unnamed member structure: 

struct person
{
    int id;
    struct {char first[20]; char last[20];};        // anonymous structure
};

You could initialize this structure in the same fashion: 

struct person jack = {123, {"James", "Bound"}};

But access is simplified as you use member names such as first as if they were person members: 

puts(jack.first);
