C Toggling Bits

PreviousNext

Toggling a bit means turning it off if it is on, and turning it on if it is off.

You can use the bitwise EXCLUSIVE OR operator to toggle a bit.

The idea is that if b is a bit setting (1 or 0), then 1 ^ b is 0 if b is 1 and is 1 if b is 0.

Also 0 ^ b is b, regardless of its value.

Therefore, if you combine a value with a mask by using ^, values corresponding to 1s in the mask are toggled, and values corresponding to 0s in the mask are unaltered.

To toggle bit 1 in flags, you can do either of the following: 

flags = flags ^ MASK;
flags ^= MASK;

For example, suppose flags is 00001111 and MASK is 10110110. The expression 

flags ^ MASK

becomes 

(00001111) ^ (10110110)      // expression

and evaluates to the following: 

(10111001)                   // resulting value

All the bits that are set to 1 in MASK result in the corresponding bits of flags being toggled.

All the bits in flags that corresponded to 0 bits in MASK are left unchanged.
