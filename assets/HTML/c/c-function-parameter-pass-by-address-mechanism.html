C Function Parameter Pass-by-Address Mechanism

PreviousNext

The only way that a called function can change the value of a variable belonging to the calling function is by receiving an argument value that's the address of the variable.

When you pass an address to a function, it's still only a copy of the address that's passed, not the original.

However, the copy is still the same address and still references the original variable.

This is the reason why you must pass the addresses of variables to scanf().

Without the addresses, the function cannot store the values in the original variables.

Example

The following code shows how to pass address to a function: 

#include <stdio.h> 

void my(int* n) 
{ 
  printf("Inside function: n is now %d \n",*n); 
  *n = 5;
  printf("Inside function: n is now %d \n",*n); 
} 
// main program - execution always starts here 
int main(void) 
{ 
  int n = 0; 
  printf("main: n is now %d \n",n); 
  my(&n); 
  printf("main: n is now %d \n",n); 

  return 0; 
}

Output:

From the result we can see that the change from the function is kept in the main() function.
