C Static Variables with Block Scope

PreviousNext

static means that the variable stays put in memory, not necessarily in value.

Variables with file scope automatically have static storage duration.

You also can create local variables having block scope but static duration.

These variables have the same scope as automatic variables, but they don't vanish when the containing function ends its job.

Such variables have block scope, no linkage, but static storage duration.

The computer remembers their values from one function call to the next.

Such variables are created by declaring them in a block, which provides the block scope and lack of linkage, with the storage-class specifier static, which provides the static storage duration.

Example

The example in illustrates this technique. 

#include <stdio.h>
void f(void);

int main(void)
{
    int count;

    for (count = 1; count <= 3; count++)
    {
        printf("count %d:\n", count);
        f();
    }

    return 0;
}

void f(void)
{
    int fade = 1;
    static int stay = 1;

    printf("fade = %d and stay = %d\n", fade++, stay++);
}

Output:

The static variable stay remembers that its value was increased by 1, but the fade variable starts a new value each time.

fade is initialized each time f() is called, but stay is initialized just once, when f() is compiled.

Static variables are initialized to zero if you don't explicitly initialize them to some other value.

The two declarations look similar: 

int fade = 1;
static int stay = 1;

The first statement is really part of the f() function and is executed each time the function is called. It is a runtime action.

The second statement isn't actually part of the f() function.

static variables and external variables are already in place after a program is loaded into memory.

Placing the statement in the f() function tells the compiler that only the f() function is allowed to see the variable; it's not a statement that's executed during runtime.

You can't use static for function parameters: 

int wontwork(static int flu);        // not allowed

Another term for a static variable with block scope is a "local static variable."
