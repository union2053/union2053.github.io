C if Statement

PreviousNext

The general form is this: 

if (expression)
      statement
Next_statement; 

If expression evaluates to true (nonzero), statement is executed.

Otherwise, it is skipped.

The statement can be either a single statement or a block compound statement.

Normally, expression is a relational expression. That is, it compares the magnitude of two quantities, as in the expressions x > y and c == 6.

If expression is true (x is greater than y, or c does equal 6), the statement is executed.

Otherwise, the statement is ignored.

More generally, any expression can be used, and an expression with a 0 value is taken to be false.

Syntax

The statement portion can be a simple statement or it can be a compound statement or block, marked off by braces: 

if (a > b)
    printf("Jackpot!\n");      // simple statement

if (c > d)
{                          // compound statement
    joecash++;
    printf("You lose, Ron.\n");
}

Example

A simple example of an if statement.

This program reads in a list of daily low temperatures (in Celsius) and reports the total number of entries and the percentage that were below freezing (that is, below zero degrees Celsius).

It uses scanf() in a loop to read in the values.

Once during each loop cycle, it increments a counter to keep track of the number of entries.

An if statement identifies temperatures below freezing and keeps track of the number of below-freezing days separately. 

#include <stdio.h>
int main(void)
{
    const int FREEZING = 0;
    float temperature;
    int cold_days = 0;
    int all_days = 0;

    printf("Enter the list of daily low temperatures.\n");
    printf("Use Celsius, and enter q to quit.\n");
    while (scanf("%f", &temperature) == 1)
    {
        all_days++;
        if (temperature < FREEZING)
            cold_days++;
    }
    if (all_days != 0)
        printf("%d days total: %.1f%% were below freezing.\n",
               all_days, 100.0 * (float) cold_days / all_days);
    if (all_days == 0)
        printf("No data entered!\n");

    return 0;
}
