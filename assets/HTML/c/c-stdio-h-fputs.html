C stdio.h fputs

PreviousNext

Example

The following code appends a string to a text file. 

#include <stdio.h>

int main ()
{
   FILE * pFile;
   char sentence [256] = "this is a test from demo2s.com";
   
   pFile = fopen ("mylog.txt","a");
   fputs (sentence,pFile);
   fclose (pFile);
   return 0;
}

Output:

Prototype 

int fputs ( const char * str, FILE * stream );

Description

The fputs() function writes string to file stream.

This terminating null-character from the string is not copied to the stream.

fputs() does not write additional characters, while puts() appends a newline character at the end automatically.

Parameters

Item
 Description
str
 C string to be written to stream.
stream
 Pointer to a FILE object

Return Value

On success, a non-negative value is returned.

On error, the function returns EOF and sets the error indicator (ferror).
