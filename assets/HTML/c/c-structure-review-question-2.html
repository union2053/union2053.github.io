Question 1

Consider the following programming fragment: 

struct name {
        char first[20];
        char last[20];
};
struct bem {
        int limbs;
        struct name title;
        char type[30];
};
struct bem * pb;
struct bem deb = {
      6,
       {"CSS", "HTML"},
       "Web"


};

pb = &deb;

a. What would each of the following statements print?
   printf("%d\n", deb.limbs);
   printf("%s\n", pb->type);

b. How could you represent "HTML" in structure notation (two ways)?


a.     6
       Web

b.     Use the structure name and use the pointer:
       deb.title.last
       pb->title.last

Question 2

Consider the following declarations: 

struct fullname {
                char fname[20];
                char lname[20];
};
struct record{
        struct fullname name;
        int born;
        int died;
};
struct record jack;
struct record *pt = &jack;


a. Identify the born member of the jack structure using the jack identifier.

b. Identify the born member of the jack structure using the pt identifier.

c. Use a scanf() call to read in a value for the born member using the jack
   identifier.

d. Use a scanf() call to read in a value for the born member using 
   the pt identifier.

e. Use a scanf() call to read in a value for the lname member of the name 
   member using the jack identifier.

f. Use a scanf() call to read in a value for the lname member of the name 
   member using the pt identifier.

g. Construct an identifier for the third letter of the first name of someone 
   described by the jack variable.

h. Construct an expression representing the total number of letters in the 
   first and last names of someone described by the jack variable.


a. jack.born

b. pt->born

c. scanf("%d", &jack.born);

d. scanf("%d", &pt->born);

e. scanf("%s", jack.name.lname);

f. scanf("%s", pt->name.lname);

g. jack.name.fname[2]

h. strlen(jack.name.fname) + strlen(jack.name.lname)

Question 3

Define a structure template suitable for holding the following items:

  1. the name of an automobile,
  2. its horsepower,
  3. its EPA city-driving MPG rating,
  4. its wheelbase, and
  5. its year.

Use car as the template tag. 


Here is one possibility:

struct car {

    char name[20];
    float hp;

    float epampg;
    float wbase;
    int year;
};

Question 4

Suppose you have this structure: 

struct gas {
        float distance;
        float gals;
        float mpg;
};

a. Devise a function that takes a struct gas argument.

Assume that the passed structure contains the distance and gals information.

Have the function calculate the correct value for the mpg member and return the now completed structure.

b. Devise a function that takes the address of a struct gas argument.

Assume that the passed structure contains the distance and gals information.

Have the function calculate the correct value for the mpg member and assign it to the appropriate member. 


The functions could be set up like this:

struct gas {
   float distance;
   float gals;
   float mpg;
};

struct gas mpgs(struct gas trip)
{
  if (trip.gals > 0)
     trip.mpg = trip.distance / trip.gals ;
  else
     trip.mpg = -1.0;
  return trip;
}

void set_mpgs(struct gas * ptrip)
{
  if (ptrip->gals > 0)
     ptrip->mpg = ptrip->distance / ptrip->gals ;
  else
     ptrip->mpg = -1.0;
}

Note that the first function cannot directly alter 
values in the calling program, so you
must use the return value to convey the information:

struct gas idaho = {430.0, 14.8};  // set first two members
idaho = mpgs(idaho);               // reset structure

The second function, however, accesses the original structure directly:

struct gas ohio = {583, 17.6};   // set first two members
set_mpgs(&ohio);                  // set third member
