C calloc() Function to allocate memory

PreviousNext

Another option for memory allocation is to use calloc().

A typical use looks like this: 

long * newmem;
newmem = (long *)calloc(100, sizeof (long));

calloc() returns a pointer-to-void under C ANSI.

You should use the cast operator if you want to store a different type.

The function takes two arguments, both of which should be unsigned integers (type size_t since ANSI).

The first argument is the number of memory cells you want.

The second argument is the size of each cell in bytes.

In the code above, long uses 4 bytes, so this instruction sets up 100 4-byte units, using 400 bytes in all for storage.

Using sizeof (long) instead of 4 makes this coding more portable. It will work on those systems where long is some size other than 4.

The calloc() function has in one more feature: It sets all the bits in the block to zero.

The free() function can also be used to free memory allocated by calloc().
