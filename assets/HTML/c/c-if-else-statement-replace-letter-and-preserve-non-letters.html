C if else statement replace letter and preserve non-letters

PreviousNext

The following program repeats an input line but replaces each letter with the letter that follows it in the ASCII code sequence.

Preserving non-letters: 

#include <stdio.h>
#include <ctype.h>            // for isalpha()
int main(void)
{
    char ch;

    while ((ch = getchar()) != '\n')
    {
        if (isalpha(ch))      // if a letter,
             putchar(ch + 1); // display next letter
        else                  // otherwise,
             putchar(ch);     // display as is
    }
    putchar(ch);              // display the newline

    return 0;
}
