C Multidimensional Arrays

PreviousNext

We can create array with more Multi-dimensions.

Here is how to declare such an array: 

float my_array[5][12];     // array of 5 arrays of 12 floats

One way to view this declaration is to first look at the inner portion:

It tells us that my_array is an array with five elements.

But what is each of those elements? Now look at the remaining part of the declaration:

This tells us that each element is of type float[12]; that is, each of the five elements of my_array is, in itself, an array of 12 float values.

my_array[0], being the first element of my_array, is an array of 12 float values.

So are my_array[1], my_array[2], and so on.

If my_array[0] is an array, its first element is my_array[0][0], its second element is my_array[0][1], and so on.

In short, my_array is a five-element array of 12-element arrays of float, my_array[0] is an array of 12 floats, and my_array[0][0] is a float.

To access, say, the value in row 2, column 3, use my_array[2][3].

You can also visualize this my_array array as a two-dimensional array consisting of five rows, each of 12 columns. 

{
    {4.3,4.3,4.2,3.0,2.0,2.2,0.2,0.2,0.4,2.4,3.5,6.6},
    {8.5,8.2,1.2,1.6,2.4,0.0,5.2,0.9,0.3,0.9,1.4,7.3},
    {9.1,8.5,6.7,4.3,2.1,0.8,0.2,0.2,1.1,2.3,6.1,8.4},
    {7.2,9.9,8.4,3.3,1.2,0.8,0.4,0.1,0.6,1.7,4.3,6.2},
    {7.6,5.6,3.8,2.8,3.9,0.2,0.1,0.1,0.0,1.3,2.6,5.2}
};

Example

The following program finds the total precipitation for each year, the average yearly precipitation, and the average precipitation for each month.

To find the total precipitation for a year, you have to add all the data in a given row.

To find the average precipitation for a given month, you have to add all the data in a given column.

The two-dimensional array makes it easy to visualize and execute these activities. 

#include <stdio.h>
#define MONTHS 12      // number of months in a year
#define YEARS    5     // number of years of data
int main(void)
{
  // initializing precipitation data for 2010 - 2014
     const float my_array[YEARS][MONTHS] =
    {
        {4.3,4.3,4.2,3.0,2.0,2.2,0.2,0.2,0.4,2.4,3.5,6.6},
        {8.5,8.2,1.2,1.6,2.4,0.0,5.2,0.9,0.3,0.9,1.4,7.3},
        {9.1,8.5,6.7,4.3,2.1,0.8,0.2,0.2,1.1,2.3,6.1,8.4},
        {7.2,9.9,8.4,3.3,1.2,0.8,0.4,0.1,0.6,1.7,4.3,6.2},
        {7.6,5.6,3.8,2.8,3.9,0.2,0.1,0.1,0.0,1.3,2.6,5.2}
    };
     int year, month;
     float subtot, total;

    printf(" YEAR    RAINFALL (inches)\n");
    for (year = 0, total = 0; year < YEARS; year++)
    {             // for each year, sum precipitation for each month
        for (month = 0, subtot = 0; month < MONTHS; month++)
            subtot += my_array[year][month];
        printf("%5d %15.1f\n", 2010 + year, subtot);
        total += subtot; // total for all years
      }
    printf("\nThe yearly average is %.1f inches.\n\n",
            total/YEARS);
    printf("MONTHLY AVERAGES:\n\n");
    printf(" Jan Feb Mar Apr May Jun Jul Aug Sep Oct ");
    printf(" Nov Dec\n");

    for (month = 0; month < MONTHS; month++)
    {             // for each month, sum precipitation over years
        for (year = 0, subtot =0; year < YEARS; year++)
            subtot += my_array[year][month];
        printf("%4.1f ", subtot/YEARS);
    }
    printf("\n");

    return 0;
}
