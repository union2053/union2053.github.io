C while loop check end of file flag

PreviousNext

The following code echo user input for each end of file flag:

To use EOF in a program, compare the return value of getchar() with EOF.

If they are different, you have not yet reached the end of a file.

In other words, you can use an expression like this: 

while ((ch = getchar()) != EOF)

You can simulate an end-of-file condition from the keyboard by pressing enter. 

#include <stdio.h>
int main(void)
{
    int ch;

    while ((ch = getchar()) != EOF)
        putchar(ch);

    return 0;
}

Note

You don't have to define EOF because stdio.h takes care of that.

You don't have to worry about the actual value of EOF, because the #define statement in stdio.h enables you to use the symbolic representation EOF.

You shouldn't write code that assumes EOF has a particular value.
