Question 1

What will this program print? 

#include <stdio.h>
int main(void)
{
  int ref[] = {8, 4, 0, 2};
  int *ptr;
  int index;

    for (index = 0, ptr = ref; index < 4; index++, ptr++)
       printf("%d %d\n", ref[index], *ptr);
    return 0;
}


The printout is this:
8 8
4 4
0 0
2 2

Question 3

In the following code, ref is the address of what? 


What about ref + 1? What does ++ref point to?


#include <stdio.h>
int main(void)
{
  int ref[] = {8, 4, 0, 2};
  int *ptr;
  int index;

    for (index = 0, ptr = ref; index < 4; index++, ptr++)
       printf("%d %d\n", ref[index], *ptr);
    return 0;
}


The array name ref  points to the first element of the array, the integer 8. 

The expression ref + 1  points to the second element, the integer 4. 

The construction  ++ref  is not a valid C expression; 
ref  is a constant, not a variable.


Question 4

What is the value of *ptr and of *(ptr + 2) in each case?

a.
int *ptr;
int torf[2][2] = {12, 14, 16};
ptr = torf[0];

b.
int * ptr;
int fort[2][2] = { {12}, {14,16} };
ptr = fort[0];


ptr  points to the first element, and  ptr + 2  points to the third element, 
which would be the first element of the second row.

a. 12 and 16.

b. 12 and 14.


Question 5

 What is the value of **ptr and of **(ptr + 1) in each case?

a.
int (*ptr)[2];
int torf[2][2] = {12, 14, 16};
ptr = torf;

b.
int (*ptr)[2];
int fort[2][2] = { {12}, {14,16} };
ptr = fort;


ptr  points to the first row and  ptr+1  points to the second row;  
*ptr  points to the first element in the first row, and *(ptr + 1)  
 points to the first element of the second row.

a. 12 and 16.

b. 12 and 14.
