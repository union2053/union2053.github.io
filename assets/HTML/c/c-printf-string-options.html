C printf string options

PreviousNext

Let's examine some of the string options.

Consider the example in the following code. 

/* stringf.c -- string formatting */
#include <stdio.h>
#define BLURB "ABCDEDFG HIJKLMN!"
int main(void)
{
    printf("[%2s]\n", BLURB);
    printf("[%24s]\n", BLURB);
    printf("[%24.5s]\n", BLURB);
    printf("[%-24.5s]\n", BLURB);

    return 0;
}

Here is the output:

For the %2s specification, the field is expanded to contain all the characters in the string.

Also notice how the precision specification limits the number of characters printed.

The .5 in the format specifier tells printf() to print just five characters.

The - modifier left-justifies the text.
