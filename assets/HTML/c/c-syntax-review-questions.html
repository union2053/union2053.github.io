Question 5

Consider the following program: 

#include <stdio.h>
int main(void)
{
  int a, b;

     a = 5;
     b = 2;    /* line 7 */
     b = a;    /* line 8 */
     a = b;    /* line 9 */
     printf("%d %d\n", b, a);
     return 0;
}

What is the program state after line 7? Line 8? Line 9? 


After line 7, a  is 5 and  b  is 2.
After line 8, both  a  and  b  are 5. After line 9, both  a  and b  are still 5.

Question 6

Consider the following program: 

#include <stdio.h>
int main(void)
{
    int x, y;

    x = 10;
    y = 5;        /* line 7 */

    y = x + y;    /* line 8 */
    x = x*y;      /* line 9 */
    printf("%d %d\n", x, y);
    return 0;
}

What is the program state after line 7? Line 8? Line 9? 


After line 7, x  is 10 and  b  is 5.
After line 8,  x  is 10 and  y  is 15.
After line 9,  x  is 150 and  y is 15.
