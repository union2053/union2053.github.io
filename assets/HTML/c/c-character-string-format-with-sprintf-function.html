C Character String format with sprintf() Function

PreviousNext

The sprintf() function is declared in stdio.h instead of string.h.

It works like printf(), but it writes to a string instead of writing to a display.

It combines several elements into a single string.

The first argument to sprintf() is the address of the target string.

The remaining arguments are the same as for printf()-a conversion specification string followed by a list of items to be written.

The following code uses sprintf() to combine three items (two strings and a number) into a single string.

It uses sprintf() the same way you would use printf(), except that the resulting string is stored in the array formal instead of being displayed on screen. 

#include <stdio.h>
#define MAX 20
char * s_gets(char * st, int n);

int main(void)
{
    char first[MAX] = "HTML";
    char last[MAX] = "CSS";
    char formal[2 * MAX + 10];
    double prize = 213.234;

    sprintf(formal, "%s, %-19s: $%6.2f\n", last, first, prize);
    puts(formal);

    return 0;
}

Output:

The sprintf() command took the input and formatted it into a standard form, which it then stored in the string formal.
