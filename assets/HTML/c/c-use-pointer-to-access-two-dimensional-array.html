C use pointer to access two-dimensional array

PreviousNext

Using two index values accesses the value stored in an element of the array.

The following code show to use my_array to get to all the values in that array. //Getting values in a two-dimensional array 

#include <stdio.h>

int main(void) 
{ 
  char my_array[3][3] = { 
                       {'1','2','3'}, 
                       {'4','5','6'}, 
                       {'7','8','9'} 
                     }; 

   // List all elements of the array 
  for(int i = 0 ; i < 9 ; ++i) 
    printf(" my_array: %c\n", *(*my_array + i)); 
  return 0; 
}

The output from the program is as follows:

You dereference my_array in the loop: 

printf(" my_array: %c\n", *(*my_array + i));

You use the expression (my_array + i) to get the value of an array element. the expression between the parentheses, *my_array + i, produces the address of the element in the my_array array that is at offset i.

We can use a variable that you've declared as a pointer.

If you declare a pointer and assign the address of the array to it, then you can use that pointer to access the members of the array by modifying the address it contains. 

#include <stdio.h> 

int main(void) 
{ 
  char my_array[3][3] = { 
                       {'1','2','3'}, 
                       {'4','5','6'}, 
                       {'7','8','9'} 
                     }; 

  char *pmy_array = *my_array;             // A pointer to char 
  for(int i = 0 ; i < 9 ; ++i) 
    printf(" my_array: %c\n", *(pmy_array + i)); 

  return 0; 
}

Output:

You initialize pmy_array with the address of the first element of the array, and then you use normal pointer arithmetic to move through the array: 

char *pmy_array = *my_array;             // A pointer to char 
for(int i = 0 ; i < 9 ; ++i) 
    printf(" my_array: %c\n", *(pmy_array + i));

You dereference my_array to obtain the address you want (with *my_array).

This is necessary because my_array by itself is of type char**, a pointer to a pointer, and is the address of the subarray my_array[0].

You could have initialized pmy_array by using the following: 

char *pmy_array = &my_array[0][0];

You might think you could initialize pmy_array using this statement: 

pmy_array = my_array;                    // Wrong level of indirection!

pmy_array and my_array have different levels of indirection.

pmy_array refers to an address that contains a value of type char, whereas my_array refers to an address that refers to an address containing a value of type char.
