C Initializing a Structure

PreviousNext

Suppose we have the following structure: 

struct {         /* no tag */
    char title[MAXTITL];
    char author[MAXAUTL];
    float value;
} library;

To initialize a structure, you use a syntax similar to that used for arrays: 

struct book library = {
    "How to use HTML",
    "demo2s.com",
    1.95
};

In short, you use a comma-separated list of initializers enclosed in braces.

Each initializer should match the type of the structure member being initialized.

Therefore, you can initialize the title member to a string and the value member to a number.

To make the associations more obvious, we gave each member its own line of initialization, but all the compiler needs are commas to separate one member's initialization from the next.
