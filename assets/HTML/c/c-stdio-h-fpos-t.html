C stdio.h fpos_t

PreviousNext

Example 

#include <stdio.h>
#include <stdlib.h>
#include <assert.h>

int main(void)
{
    FILE* fp = fopen("main.c", "r");
    // demo using fsetpos to return to the beginning of a file
    fpos_t pos;
    fgetpos(fp, &pos);               // store start of file in pos
    fclose(fp);
}

Description

This type of object fpos_t contains information about a position within a file.

It can record any position within a file.

The fpos_t objects is filled by a call to fgetpos().

The fpos_t object is used directly, but only used as an argument in a call to fsetpos().
