C Character String output with puts() function

PreviousNext

The puts() function is very easy to use.

Just give it the address of a string for an argument.

The following code illustrates some ways to use puts() function. 

#include <stdio.h>
#define DEF "I am a #defined string."
int main(void)
{
    char str1[80] = "char array.";
    const char * str2 = "A pointer";

    puts("an argument to puts().");
    puts(DEF);
    puts(str1);
    puts(str2);
    puts(&str1[5]);
    puts(str2+4);

    return 0;
}

Output:

Each string appears on its own line because puts() automatically appends a newline when it displays a string.

The text in double quotation marks are string constants and are treated as addresses.

The names of character array strings are treated as addresses.

The expression &str1[5] is the address of the sixth element of the array str1.

That is what puts() uses for its starting point.

puts() stops when it encounters the null character.
