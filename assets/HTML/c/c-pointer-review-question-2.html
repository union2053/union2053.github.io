Question 1

Suppose you have the following declaration: 

int my_array[30][100];.

(a) Express the address of my_array[22][56] one way.

(b) Express the address of my_array[22][0] two ways.

(c) Express the address of my_array[0][0] three ways. 


a.&my_array[22][56]
  &my_array[22][0] or my_array[22]

b. &my_array[22][0] or my_array[22]

The latter is the name of a one-dimensional array of 100 elements, 
hence the address of its first element, which is the element my_array[22][0].

c.&my_array[0][0] or my_array[0] or (int *) my_array

Here,  my_array[0]  is the address of the  int  element  my_array[0][0] , 
and  my_array  is the address of the 100-element array my_array[0]. 

The two addresses have the same numeric value but
different types; the typecast makes the types the same.

Question 2

Create an appropriate declaration for each of the following variables: 

a. digits is an array of 10 ints.

b. rates is an array of six floats.

c. mat is an array of three arrays of five integers.

d. psa is an array of 20 pointers to char.

e. pstr is a pointer to an array of 20 chars.


a. int digits[10];

b. float rates[6];

c. int mat[3][5];

d. char * psa[20] ;

Note that [] has higher precedence than *, so in the absence of parentheses, 
the array descriptor is applied first, and then the pointer descriptor. 

Hence, this declaration is the same as char *(psa[20]); .
                     
e. char (*pstr)[20];

Question 3

Suppose you have these declarations: 

float my_array[10], things[10][5], *pf, value = 2.2;
int i = 3;

Identify each of the following statements as valid or invalid: 

a. my_array[2] = value;

b. scanf("%f", &my_array );

c. my_array = value;

d. printf("%f", my_array);

e. things[4][4] = my_array[3];

f. things[5] = my_array;

g. pf = value;

h. pf = my_array;


a. my_array[2] = value;
Valid.

b. scanf("%f", &my_array );
Invalid;  my_array  is not a  float .

c. my_array = value;
Invalid;
my_array  is not a  float .

d. printf("%f", my_array);
Invalid;  my_array  is not a  float .

e. things[4][4] = my_array[3];
Valid.

f. things[5] = my_array;
Invalid; can't assign arrays.

g. pf = value;
Invalid;  value  is not an address.

h. pf = my_array;
Valid.

Question 5

Here are two function prototypes: 

void show(const double ar[], int n);   // n is number of elements
void show2(const double ar2[][3], int n); // n is number of rows

a. Show a function call that passes a compound literal containing 
   the values 8, 3, 9, and 2 to the show() function.
b. Show a function call that passes a compound literal containing 
   the values 8, 3, and 9 as the first row and the values 5, 4, 
   and 1 as the second row to the show2() function.


a. show( (int [4]) {8,3,9,2}, 4);
b. show2( (int [][3]){{8,3,9}, {5,4,1}}, 2);
