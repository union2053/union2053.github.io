C Expression Value

PreviousNext

Every C expression has a value.

To find the value, you perform the operations in the order dictated by operator precedence.

The expressions with = simply have the same value that the variable to the left of the = sign receives.

Therefore, the expression q = 5 * 2 as a whole has the value 10.

The expression q > 3, relational expressions, has the value 1 if true and 0 if false.

Here are some expressions and their values:

Expression Value
-4 + 6 2
c = 3 + 811
5 > 31
6 + (c = 3 + 8) 17

The last expression is perfectly legal in C because it is the sum of two subexpressions, each of which has a value.
