C stdio.h fputc

PreviousNext

Example

The following code uses fputc() function to write alphabet to a text file. 

#include <stdio.h>

int main ()
{
  FILE * pFile;

  pFile = fopen ("main.c","w");
  if (pFile!=NULL) {
    for (char c = 'A' ; c <= 'Z' ; c++)
      fputc ( c , pFile );

    fclose (pFile);
  }
  return 0;
}

Output:

Prototype 

int fputc ( int character, FILE * stream );

Description

fputc() functions writes character to file stream and advances the position indicator.

Parameters

Item
 Description
character

 The character to be written.
The value is converted to an unsigned char when written.
stream
 Pointer to a FILE object.

Return Value

On success, the character written is returned.

If a writing error occurs, EOF is returned and the error indicator (ferror) is set.
