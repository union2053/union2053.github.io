C stdio.h fscanf

Previous

Example 

#include <stdio.h>

int main ()
{
  char str [80];
  float f;
  FILE * pFile;

  pFile = fopen ("main.c","w+");
  fprintf (pFile, "%f %s", 3.1416, "PI");
  rewind (pFile);
  fscanf (pFile, "%f", &f);
  fscanf (pFile, "%s", str);
  fclose (pFile);
  printf ("I have read: %f and %s \n",f,str);
  return 0;
}

This sample code creates a file called main.c and writes a float number and a string to it.

Then, the stream is rewinded and both values are read with fscanf.

Prototype 

int fscanf ( FILE * stream, const char * format, ... );

Description

The function fscanf() can read formatted data from a file stream.

It uses the same format string as scanf();

Parameters

Item
 Description
stream
 Pointer to a FILE object
format
 string that contains a sequence of control characters
...

 additional arguments
Depending on the format string

Return Value

On success, the function returns the number of items of the argument list successfully filled.

If a reading error happens or the end-of-file is reached while reading, the proper indicator is set (feof or ferror).

EOF is returned if reaches the file end.
