C Use Bitwise Operator to convert letter to lowercase

PreviousNext

Suppose letter is of type char.

Convert a letter to lowercase: 

letter =  (letter | ' ');

The trick is that space ASCII value is 32 since 'a' lowercase is 65 and uppercase is 97 (97 - 65 = 32). 

#include <stdio.h> 

int main(void) 
{ 

    char letter = 'C'; 
    letter =  (letter | ' '); 
    
    printf("%c",letter);
    return 0; 
}

Note

There are bitwise techniques to convert characters to lowercase/uppercase or invert between those states:

Invert text case 

letter ^= ' ';

Convert to lowercase: 

letter |= ' ';

Convert to uppercase; 

letter &= '_';
