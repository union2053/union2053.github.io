C printf() Character Output

PreviousNext

There are two flavors of output specifications you can use with printf() for character data: %c for single characters and %s for strings.

%lc and %ls are used to output wide characters.

Outputting character data 

#include <stdio.h> 
#include <limits.h>               // For CHAR_MAX 
#include <ctype.h>                // For isprint() 

int main(void) 
{ 
  int count = 0; 

  printf("The printable characters are the following:\n"); 

  // Iterate over all values of type char 

  for(int code = 0 ; code <= CHAR_MAX ; ++code) 
  { 
    char ch = (char)code; 
    if(isprint(ch)) 
    { 
      if(count++ % 32 == 0) 
        printf("\n"); 
      printf(" %c", ch); 
    } 
  } 
  printf("\n"); 
  return 0; 
}

Output:

The isprint() function is declared in the ctype.h header to test for a printable character.

When isprint() returns true, you output the character using the %c format specification.
