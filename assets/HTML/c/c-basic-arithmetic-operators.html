C Basic Arithmetic Operators

PreviousNext
Operator Action
+Addition
- Subtraction
* Multiplication
/ Division
% Modulus

The items of data that an operator applies to are generally referred to as operands.

All these operators produce an integer result when both operands are integers.

modulus

The modulus operator calculates the remainder after dividing the value of the expression on the left of the operator by the value of the expression on the right.

It's sometimes referred to as the remainder operator.

The expression 12 % 5 produces 2, because 12 divided by 5 leaves a remainder of 2.

Important: All these operators work as you'd expect, with the exception of division. When it is applied to integers the result is also integer.

Example 

#include <stdio.h> 

int main(void) 
{ 
  int beans = 5; 
  int bean_calories = 125;                // Calories per bean 
  int total_eaten = 0;                      // Total beans eaten 

  int eaten = 2;                            // Number to be eaten 
  beans = beans - eaten;                // Subtract number eaten from beans 
  total_eaten = total_eaten + eaten; 
  printf("\nI have eaten %d beans.  There are %d beans left", 
                                                       eaten, beans); 

  eaten = 3;                                // New value for beans eaten 
  beans = beans - eaten;                // Subtract number eaten from beans 
  total_eaten = total_eaten + eaten; 
  printf("\nI have eaten %d more. Now there are %d beans left\n", 
            eaten, beans); 
  printf("\nTotal energy consumed is %d calories.\n", 
            total_eaten*bean_calories); 
  return 0; 
}

Note

You first declare and initialize three variables of type int: 

int beans = 5;
int bean_calories = 125;                // Calories per bean
int total_eaten = 0;                      // Total beans eaten

You'll use the total_eaten variable to accumulate the total number of beans eaten as the program progresses, so you initialize it to 0.

Next, you declare and initialize a variable that holds the number of beans to be eaten: 

int eaten = 2;                            // Number to be eaten

You use the subtraction operator to subtract eaten from the value of beans: 

beans = beans - eaten;                // Subtract number eaten from beans

The result of the subtraction is stored back in the variable beans, so the value of beans will now be 3.

Because you've eaten some beans, you increment the count of the total that you've eaten by the value of eaten: 

total_eaten = total_eaten + eaten;

You add the current value of eaten, which is 2, to the current value of total_eaten, which is 0. the result is stored back in the variable total_eaten.

The printf() statement displays the number of beans that have been eaten and that are left: 

printf("\nI have eaten %d beans.  There are %d beans left",eaten, beans);

When you need to split a string over two or more lines, each segment of the string on a line must have its own pair of double quotes delimiting it.

For example, you could write the previous statement as follows: 

printf("\nI have eaten %d beans. " 
          " There are %d beans left", 
          eaten, beans);

Where there are two or more strings immediately following one another like this, the compiler will join them to form a single string.

You display the values stored in eaten and beans using the conversion specifier, %d, for integer values.

The value of eaten will replace the first %d in the output string, and the value of beans will replace the second.

The string is displayed starting on a new line because of the \n at the beginning. the next statement sets the variable eaten to a new value: 

eaten = 3;                                // New value for beans to be eaten

The new value replaces the previous value stored in eaten, which was 2. You then go through the same sequence of operations as you did before: 

beans = beans - eaten;                // Subtract number eaten from beans
total_eaten = total_eaten + eaten;
printf("\nI have eaten %d more.  Now there are %d beans left\n", eaten, beans);

Finally, before executing the return statement that ends the program, you calculate and output the number of calories corresponding to the number of beans eaten: 

printf("\nTotal energy consumed is %d calories.\n", total_eaten*bean_calories);

Here the second argument to the printf() function is an arithmetic expression rather than just a variable.
