C Exercise Simple Print statement

PreviousNext

Write a program that uses one printf() call to print your first name and last name on one line.

Uses a second printf() call to print your first and last names on two separate lines

Uses a pair of printf() calls to print your first and last names on one line.

The output should look like this: 

James Bond //First print statement
James      //Second print statement
Bond       //Still the second print statement
James Bond //Third and fourth print statements


/* Programming Exercise 2-1  */
#include <stdio.h>
int main(void)
{
    printf("James Bond\n");
    printf("James\nBond\n");
    printf("James ");
    printf("Bond\n");
    return 0;
}

Question 3

Write a program that converts your age in years to days and displays both values.

Don't worry about fractional years and leap years. 


/* this converts my age in years to days */
#include <stdio.h>

int main(void)
{
  int years = 19;
  int days = years * 365;

  printf("I am %d years old, I have spent approximately %d days on earth.\n",
         years, days);

  return 0;
}

Question 4

Write a program that produces the following output: 

I like C.
I like C.
I like C.
Have a nice day!

  • one named myFunc() that prints the 'I like C.' message once,
  • and one named myFunc2() that prints the final line once.

#include <stdio.h>

void myFunc(void);
void myFunc2(void);

int main(void) /* display some lyrics */
{
  myFunc();
  myFunc();
  myFunc();
  myFunc2();

  return 0;
}

void myFunc(void)
{
  printf("I like C.\n");
}

void myFunc2(void)
{
  printf("Have a nice day!\n");
}

Question 5

Write a program that produces the following output: 

CSS, Client, HTML, Web
HTML, Web,
CSS, Client

  • one named br() that prints "CSS, Client" once, and
  • one named ic() that prints "HTML, Web" once.

Let main() take care of any additional printing tasks. 


#include <stdio.h>

void br(void);
void ic(void);

int main(void) /* display some country names */
{
  br();
  printf(", ");
  ic();
  printf("\n");
  ic();
  printf(",\n");
  br();
  printf("\n");

  return 0;
}

void br(void)
{
  printf("CSS, Client");
}

void ic(void)
{
  printf("HTML, Web");
}
