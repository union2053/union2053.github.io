C Pointers and Multidimensional Arrays

PreviousNext

Suppose you have this declaration: 

int my_array[4][2];      /* an array of arrays of ints */

Rule 1

Then my_array, being the name of an array, is the address of the first element of the array.

In this case, the first element of my_array is itself an array of two ints, so my_array is the address of an array of two ints.

Because my_array is the address of the array's first element, my_array has the same value as &my_array[0].

Next, my_array[0] is itself an array of two integers, so my_array[0] has the same value as &my_array[0][0], the address of its first element, an int.

In short, my_array[0] is the address of an int-sized object, and my_array is the address of a two-int-sized object.

Because both the integer and the array of two integers begin at the same location, both my_array and my_array[0] have the same numeric value.

Rule 2

Adding 1 to a pointer or address yields a value larger by the size of the referred-to object.

In this respect, my_array and my_array[0] differ, because my_array refers to an object two ints in size, and my_array[0] refers to an object one int in size.

Therefore, my_array + 1 has a different value from my_array[0] + 1.

Rule 3

Dereferencing a pointer or an address (applying the * operator or else the [] operator with an index) yields the value represented by the referred-to object.

Because my_array[0] is the address of its first element, (my_array[0][0]), *(my_array[0]) represents the value stored in my_array[0][0], an int value.

Similarly, *my_array represents the value of its first element, my_array[0], but my_array[0] itself is the address of an int.

It's the address &my_array[0][0], so *my_array is &my_array[0][0].

Applying the dereferencing operator to both expressions implies that **my_array equals *&my_array[0][0], which reduces to my_array[0][0], an int.

In short, my_array is the address of an address and must be dereferenced twice to get an ordinary value.

An address of an address or a pointer of a pointer is an example of double indirection.

The following code illustrates the ideas talked above. 

#include <stdio.h>
int main(void)
{
    int my_array[4][2] = { {2,4}, {6,8}, {1,3}, {5, 7} };

    printf("    my_array = %p,    my_array + 1 = %p\n",
                my_array,         my_array + 1);
    printf("my_array[0] = %p, my_array[0] + 1 = %p\n",
             my_array[0],      my_array[0] + 1);
    printf(" *my_array = %p,     *my_array + 1 = %p\n",
               *my_array,        *my_array + 1);
    printf("my_array[0][0] = %d\n", my_array[0][0]);
    printf(" *my_array[0] = %d\n", *my_array[0]);
    printf("     **my_array = %d\n", **my_array);
    printf("       my_array[2][1] = %d\n", my_array[2][1]);
    printf("*(*(my_array+2) + 1) = %d\n", *(*(my_array+2) + 1));

    return 0;
}

Output:

Note

The output shows that the address of the two-dimensional array, my_array, and the address of the one-dimensional array, my_array[0], are the same.

Each is the address of the corresponding array's first element, and this is the same numerically as &my_array[0][0].

The program shows that my_array[0] and *my_array are identical, and they should be.

Next, it shows that the name of a two-dimensional array has to be dereferenced twice to get a value stored in the array.

This can be done by using the indirection operator (*) twice or by using the bracket operator ([ ]) twice.

In particular, note that the pointer notation equivalent of my_array[2][1] is ((my_array+2) + 1).

You probably should make the effort at least once in your life to break this down.

Let's build up the expression in steps: 

my_array      //the address of the first two-int element
my_array+2    //the address of the third two-int element
*(my_array+2) //the third element, a two-int array, hence the address 
              // of its first element, an int
*(my_array+2) + 1     //the address of the second element of the 
                      // two-int array, also an int
*(*(my_array+2) + 1)  //the value of the second int in the third 
                      //row (my_array[2][1])
