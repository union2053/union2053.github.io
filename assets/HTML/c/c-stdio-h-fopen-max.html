Example

The following code prints out the value of FOPEN_MAX. 

#include <stdio.h>      /* printf */

int main ()
{
  printf ("FOPEN_MAX is %d \n", FOPEN_MAX);
  return 0;
}

Description

This macro constant FOPEN_MAX defines the limit of simultaneous open files.
