C Data Type Introduction Keywords

PreviousNext

Some types of data are numbers. Some are letters or characters.

C recognizes several fundamental data types.

If a datum is a constant, the compiler can usually tell its type just by the way it looks: 42 is an integer, and 42.100 is floating point.

A variable needs to have its type announced in a declaration statement.

The following listed the fundamental type keywords.

The C90 standard added two to the list.

The C99 standard adds yet another three.

The int keyword provides the basic class of integers used in C.

The next three keywords (long, short, and unsigned ) and the C90 addition signed are used to provide variations of the basic type.

For example, 

unsigned short int myVariable = 1;
long long int myVariable2 = 1;

Next, the char keyword designates the type used for letters of the alphabet and for other characters, such as # , $ , % , and *.

The char type also can be used to represent small integers.

Next, float , double , and the combination long double are used to represent numbers with decimal points.

The _Bool type is for Boolean values ( true and false ), and _Complex and _Imaginary represent complex and imaginary numbers, respectively.

The types created with these keywords can be divided into two families on the basis of how they are stored in the computer: integer types and floating-point types.
