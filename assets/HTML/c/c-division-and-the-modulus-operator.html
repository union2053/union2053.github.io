C Division And The Modulus Operator

PreviousNext

Suppose you have a jar of 45 beans and a group of seven children.

You'll divide the beans equally among the children and work out how many each child has.

Then you'll work out how many beans are left over: 

#include <stdio.h> 

int main(void) 
{ 
  int beans = 45;                // Number of beans in the jar 
  int children = 7;              // Number of children 
  int beans_per_child = 0;       // Number of beans per child 
  int beans_left_over = 0;       // Number of beans left over 

  // Calculate how many beans each child gets when they are divided up 
  beans_per_child = beans/children;     // Number of beans per child 
  printf("You have %d children and %d beans\n", children, beans); 
  printf("Give each child %d beans.\n", beans_per_child); 

  // Calculate how many beans are left over 
  beans_left_over = beans%children; 
  printf("There are %d beans left over.\n", beans_left_over); 
  return 0; 
}

Note

Four integer variables, beans, children, beans_per_child, and beans_left_over, are declared and initialized with the following statements: 

int beans = 45;                         // Number of beans in the jar
int children = 7;                         // Number of children
int beans_per_child = 0;                // Number of beans per child
int beans_left_over = 0;                // Number of beans left over

The number of beans is divided by the number of children by using the division operator / to produce the number of beans given to each child: 

beans_per_child = beans/children;     // Number of beans per child

The next two statements output what is happening, including the value stored in beans_per_child: 

printf("You have %d children and %d beans\n", children, beans);
printf("Give each child %d beans.\n", beans_per_child);

You calculate the remainder in the next statement by using the modulus operator: 

beans_left_over = beans%children;

The expression to the right of the assignment operator calculates the remainder that results when the value of beans is divided by the value of children.

Finally, you output the remainder in the last statement: 

printf("There are %d beans left over.\n", beans_left_over);
