C Join four strings into a single string using strnlen_s() and strncat_s()

PreviousNext

The following example assembles four strings into a single string: 

// Make optional versions of functions available 
#define __STDC_WANT_LIB_EXT1__ 1  
#include <string.h>         // Header for string functions 
#include <stdio.h> 

int main(void) 
{ 
  char preamble[] = "Test:\n\n"; 
  char str[][60] = { "Message 1.\n", 
                     "Message 2?\n", 
                     "Message 3.\n", 
                     "Message 4\n", 
                     "Message 5\n" 
                   }; 

  unsigned int strCount = sizeof(str)/sizeof(str[0]); 

  // Find the total length of all the strings in str 
  unsigned int length = 0; 
  for(unsigned int i = 0 ; i < strCount ; ++i) 
    length += strnlen_s(str[i], sizeof(str[i])); 

  // Create array to hold all strings combined 
  char my_string[length + strnlen_s(preamble, sizeof(preamble)) + 1]; 

  if(strncpy_s(my_string, sizeof(my_string), preamble, sizeof(preamble))) 
  { 
    printf("Error copying preamble to my_string.\n"); 
    return 1; 
  } 

  // Concatenate strings in my_string 
  for(unsigned int i = 0 ; i < strCount ; ++i) 
  { 
    if(strncat_s(my_string, sizeof(my_string), str[i], sizeof(str[i]))) 
    { 
      printf("Error copying string str[%u].", i); 
      return 2; 
    } 
  } 

  printf("%s", my_string);  
  return 0; 
}

Note

The definition for the __STDC_WANT_LIB_EXT1__ symbol at the beginning of the source file.

Without this, the optional string functions would not be accessible.

The code uses strnlen_s() to obtain the number of characters in each string.

The second argument is the length of the array containing a string, and it prevents the function from accessing memory beyond the last element.
