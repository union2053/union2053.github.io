C stdio.h FILENAME_MAX

PreviousNext

Example

The following code prints out the value of FILENAME_MAX. 

#include <stdio.h>      /* printf */

int main ()
{
  printf ("FILENAME_MAX is %d \n", FILENAME_MAX);
  return 0;
}

Output:

Description

This macro constant FILENAME_MAX defines the maximum length of file names.

It is the size needed for an array of char elements to hold the longest file name allowed by the library.
