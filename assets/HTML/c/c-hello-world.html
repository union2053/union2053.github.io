More advanced

Let's take a look at a simple C program. 

#include <stdio.h>
int main(void)                /* a simple program             */
{
    int num;                  /* define a variable called num */
    num = 1;                  /* assign a value to num        */

    printf("Hello world "); /* use the printf() function    */
    printf("hi.\n");
    printf("My favorite number is %d because it is first.\n",num);

    return 0;
}

This program will print something on your screen.

First, use your favorite editor to create a file containing the text from the code above.

Save the file that ends in .c. For example, test.c.

Compile and run the program.

If all went well, the output should look like the following:

If the the output for this program briefly flash on screen and then disappear, you can supply extra code to make the window stay open until you strike a key.

One way is to add the following line before the return statement: 

getchar();

This code causes the program to wait for a keystroke, so the window remains open until you press a key.

Example code Explained

#include is preprocessor instructions 

#include <stdio.h>    include another file

This line tells the compiler to include the information found in the file stdio.h, which is a standard part of all C compiler packages; this file provides support for keyboard input and for displaying output. 

int main(void)  //function name

C programs consist of one or more functions, the basic modules of a C program. This program consists of one function called main.

The parentheses identify main() as a function name.

int indicates that the main() function returns an integer, and the void indicates that the main() doesn't take any arguments.

Both int and void as part of the standard ANSI C way for defining main(). 

/* a simple program */    //a comment

The symbols /* and */ enclose comments-remarks that help clarify a program.

They are intended for the reader only and are ignored by the compiler. 

{       //beginning of the body of the function

This opening brace marks the start of the statements that make up the function.

A closing brace } marks the end of the function definition. 

int num;        //a declaration statement

This statement announces that you are using a variable called num and that num will be an integer type. 

num = 1;    //an assignment statement

The statement num = 1; assigns the value 1 to the variable called num. 

printf("Hello world ");    //a function call statement

The first statement using printf() displays the phrase "Hello world" on your screen.

Here printf() is part of the standard C library. 

printf("hi.\n");    //another function call statement

The next call to the printf() function displays the last phrase.

The \n is code telling the computer to start a new line-that is, to move the cursor to the beginning of the next line. 

printf("My favorite number is %d because it is first.\n", num);

The last use of printf() prints the value of num, which is 1, embedded in the phrase in %d. It instructs the computer where and in what form to print the value of num. 

return 0;   //a return statement

A C function can return a number to the agency that used it. For the present, just regard this line as the appropriate closing for a main() function. 

} //the end

The program ends with a closing brace.
