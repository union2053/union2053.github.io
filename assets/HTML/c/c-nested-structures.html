C Nested Structures

PreviousNext

One structure can contain, or nest, another structure.

The following code shows how to create nested structure: 

#include <stdio.h>
#define LEN 20
const char * msgs[5] =
{
    "A",
    "B",
    "C",
    "D",
    "E"
};

struct names {                          // first structure
     char first[LEN];
     char last[LEN];
};

struct guy {                         // second structure
    struct names handle;             // nested structure
    char favfood[LEN];
    char job[LEN];
    float income;
};

int main(void)
{
    struct guy fellow = {    // initialize a variable
        { "F1", "F2" },
        "F3", "F4", 
        6.00
    };

     printf("Dear %s, \n\n", fellow.handle.first);
     printf("%s%s.\n", msgs[0], fellow.handle.first);
     printf("%s%s\n", msgs[1], fellow.job);
     printf("%s\n", msgs[2]);
     printf("%s%s%s", msgs[3], fellow.favfood, msgs[4]);
     return 0;
}

Note

The declaration says that handle is a variable of the struct names type. Of course, the file should also include the declaration for the names structure.

Second, to gain access to a member of a nested structure, use the dot operator twice: 

printf("Hello, %s!\n", fellow.handle.first);
