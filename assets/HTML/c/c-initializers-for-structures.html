C Initializers for Structures

PreviousNext

C99 and C11 provide designated initializers for structures.

The syntax is similar to that for designated initializers for arrays.

However, designated initializers for structures use the dot operator and member names instead of brackets and indices to identify particular elements.

For example, 

struct {         /* no tag */
    char title[MAXTITL];
    char author[MAXAUTL];
    float value;
} library;

to initialize just the value member of a book structure, you would do this: 

struct book b1 = { .value = 10.99};

You can use designated initializers in any order: 

struct book b2 = { .value = 25.99,
                   .author = "demo2s.com",
                   .title = "Tutorial for HTML"};

The last value supplied for a particular member is the value it gets.

For example, consider this declaration: 

struct book b3 = { .value = 18.90,
                    .author = "demo2s.com",
                     0.25};

The value 0.25 is assigned to the value member because it is the one immediately listed after the author member in the structure declaration.

The new value of 0.25 supersedes the value of 18.90 provided earlier.
