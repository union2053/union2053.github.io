c. #define Q "Hi."

Question 2

What changes could you make so that string Q from above code is printed out enclosed in double quotation marks? 


Use  \" , as in the following:
printf("\"%s\"\nhas %d characters.\n", Q, strlen(Q));

Question 3

Fix all the errors 

define B Hi
define X 10
main(int)
{
   int age;
   char name;

    printf("Please enter your first name.");
    scanf("%s", name);
    printf("All right, %c, what's your age?\n", name);
    scanf("%f", age);
    xp = age + X;
    printf("That's a %s! You must be at least %d.\n", B, xp);
    rerun 0;
}


#include <stdio.h>   /* don't forget this    */
#define B "Hi"   /* add #, quotes        */
#define X 10         /* add #                */
int main(void)       /* instead of main(int) */
{
   int age;
   int xp;          /* declare all variables */
   char name[40];   /* make into an array    */

   printf("Please enter your first name.\n"); /* \n for readability */
   scanf("%s", name);
   printf("All right, %s, what's your age?\n", name); /* %s for string */
   scanf("%d", &age);       /* %d, not %f, &age, not age */
   xp = age + X;
   printf("That's a %s! You must be at least %d.\n", B, xp);
   return 0;                /* not rerun                 */
}

Suppose a program starts as follows: 

#define BOOK "War and Peace"
int main(void)

The  %%  construction for printing  % .

Question

What conversion specification would you use to print each of the following?

  • a. A decimal integer with a field width equal to the number of digits
  • b. A hexadecimal integer in the form 8A in a field width of 4
  • c. A floating-point number in the form 123.123 with a field width of 10
  • d. A floating-point number in the form 2.12e+002 with a field width of 12
  • e. A string left-justified in a field of width 30

a.      %d
b.      %4X
c.      %10.3f
d.      %12.2e
e.      %-30s
