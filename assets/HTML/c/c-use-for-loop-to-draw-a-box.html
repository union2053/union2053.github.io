C Use for loop to draw a box

PreviousNext

Suppose that you want to draw a box on the screen using +.

You can use a for loop to draw a box: 

#include <stdio.h> 

int main(void) 
{ 
  printf("\n++++++++++++++");         // Draw the top of the box 

  for(int count = 1 ; count <= 8 ; ++count) 
    printf("\n+            +");       // Draw the sides of the box 

  printf("\n++++++++++++++\n");       // Draw the bottom of the box 
  return 0; 
}

The output for this program looks like this:

Note

The first printf() statement outputs the top of the box: 

printf("\n++++++++++++++");     // Draw the top of the box

the next statement is the for loop: 

for(int count = 1 ; count <= 8 ; ++count) 
      printf("\n+            +");   // Draw the sides of the box

This repeats the printf() statement eight times to output the sides of the box.

The program execution will then continue by executing the statement that follows the loop: 

printf("\n++++++++++++++\n");   // Draw the bottom of the box
