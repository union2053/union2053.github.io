Question 1

Write a program that creates an integer variable called toes.

Have the program set toes to 10.

Also have the program calculate what twice toes is and what toes squared is.

The program should print all three values, identifying them. 


/* do some calculations on toes */
#include <stdio.h>

int main(void)
{
  int toes = 10;
  int twice_toes = 2 * toes;
  int square_toes = toes * toes;

  printf("toes = %d, twice toes = %d, squared toes = %d\n",
       toes, twice_toes, square_toes);

  return 0;
}

Question 2

Write a program that produces the following output: 

Smile!Smile!Smile!
Smile!Smile!
Smile!

Have the program define a function that displays the string Smile! once, and have the program use the function as often as needed. 


#include <stdio.h>

void smile(void);

int main(void) /* this program share smiles */
{
  smile();
  smile();
  smile();
  printf("\n");
  smile();
  smile();
  printf("\n");
  smile();
  printf("\n");
}

void smile(void)
{
  printf("Smile!");
}

Question 3

In C, one function can call another.

Write a program that calls a function named one_three().

This function should display the word one on one line,

And also make a call to a second function named two(), and then display the word three on one line.

The function two() should display the word two on one line.

The main() function should display the phrase 

starting now:

Before calling one_three() and display done! after calling it.

Thus, the output should look like the following: 

starting now:
one
two
three
done!


#include <stdio.h>

void one_three(void);
void two(void);

int main(void) /* count from one to three */
{
  printf("starting now:\n");
  one_three();
  printf("done!\n");

  return 0;
}

void one_three(void)
{
  printf("one\n");
  two();
  printf("three\n");
}

void two(void)
{
  printf("two\n");
}
