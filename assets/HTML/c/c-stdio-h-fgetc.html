C stdio.h fgetc

PreviousNext

Example

The following code reads a file and check if the file contains a ; letter. 

#include <stdio.h>
int main ()
{
  FILE * pFile;
  int c;
  int n = 0;
  pFile=fopen ("main.c","r");
  if (pFile==NULL) 
      perror ("Error opening file");
  else
  {
    do {
      c = fgetc (pFile);
      if (c == ';') n++;
    } while (c != EOF);
    fclose (pFile);
    printf ("The file contains %d ; characters.\n",n);
  }
  return 0;
}

Output:

Prototype 

int fgetc ( FILE * stream );

Description

fgetc() reads a character from stream.

Parameters

Item
 Description
stream
 Pointer to a FILE object that identifies an input stream.

Return Value

On success, the character read is returned as an int value.

EOF indicates failure.

Use ferror() to check the error.
