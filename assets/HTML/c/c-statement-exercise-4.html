Question 1

Jack invests $100 at 10% simple interest.

Every year, the investment earns an interest equal to 10% of the original investment.

Tom invests $100 at 5% interest compounded annually.

Interest is 5% of the current balance, including previous addition of interest.

Write a program that finds how many years it takes for the value of Tom's investment to exceed the value of Jack's investment.

Also show the two values at that time. 


#include <stdio.h>

int main(void)
{
  float jack_value = 100;
  float tom_value = 100;
  int year = 0;

  while (tom_value <= jack_value) {
    jack_value += 10;
    tom_value *= 1.05;
    year++;
  }
  printf("Year = %d\n", year);
  printf("jack: %f\n", jack_value);
  printf("tom: %f\n", tom_value);
  return 0;
}

Question 2

Jack won a million dollars, which he places in an account that earns 8% a year.

On the last day of each year, Jack withdraws $100,000.

Write a program that finds out how many years it takes for Jack to empty his account. 


#include <stdio.h>

int main(void)
{
  double account = 1e6;
  int year = 0;
  while (account > 0) {
    account *= 1.08;
    account -= 100000;
    year++;
  }
  printf("%d\n", year);
  return 0;
}

Question 18

Jack joined a group. Initially he had five friends.

He noticed that his friend count grew in the following fashion.

The first week one friend dropped out and the remaining number of friends doubled.

The second week two friends dropped out and the remaining number of friends doubled.

In general, in the Nth week, N friends dropped out and the remaining number doubled.

Write a program that computes and displays the number of friends each week.

The program should continue until the count exceeds 150. 


#include <stdio.h>

const int MY_NUMBER = 150;

int main(void)
{
  int week = 0;
  int friend_count = 5;
  
  printf(" week | friends\n"
         "------+---------\n");
  while (friend_count <= MY_NUMBER) {
    friend_count -= ++week;
    friend_count *= 2;
    printf("%5d |%8d\n", week, friend_count);
  }
  return 0;
}
